Lab 5: Functions and Graphics

------------------
DESCRIPTION: 
This lab implements MARS MIPS Simulator to create functions that will allow users to change the background color of the display, and “draw” horizontal and vertical lines on the display.
------------------
FILES

-
Lab5.asm

This file is the interactive code file that when opened in MARS allows the user to create the vertical lines.
-

INSTRUCTIONS:
Download the file and run MARS and Open Lab5.asm. Then open the lab test file and it will call the subroutines and macros to create an image in the bitmap display using lines. It also returns what color the pixel is, in hex format.
